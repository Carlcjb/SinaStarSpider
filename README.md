# 项目简介

新浪新闻明星板块爬虫,透过几万条数据看谁是年度最火明星。

# 运行示意

保存下来的 30000 条数据（数据是从某一天截至至代码运行的那一天，这个某一天可以是代码运行的那一天的前任意天）

分析明星热度，将明星分级，并将分级结果保存至数据库

![](pic/2.png)
